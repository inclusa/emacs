Podemos utilizar el resaltado escribiendo los códigos:

```
*bold* ==> bold
~code~ ==> code
+strike+ ==> strike
/italic/ ==> italic
=verbatim= ==> code
_underline_ ==> underline
```

Para crear superíndides:

```
elemento^superínidice
```
2^4^ `2^4`

```
elemento_subíndice
```

H~2~ `H_2`
