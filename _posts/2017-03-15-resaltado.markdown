Podemos utilizar el resaltado escribiendo los códigos:

```emacs
*bold* ==> bold
~code~ ==> code
+strike+ ==> strike
/italic/ ==> italic
=verbatim= ==> code
_underline_ ==> underline
```

Para crear superíndides:

```emacs
elemento^superínidice
```
2<sup>4</sup> `2^4`

```emacs
elemento_subíndice
```

H<sub>2</sub>O `H_2`
